AI Conversation Example:

You: Create a modern logo for "TechCorp" with a blue circle background and white text, then add a drop shadow effect
Assistant: I'll create a professional logo design for TechCorp with the specifications you mentioned.

Complete Workflow:

// Full logo creation process:
1. affinity_new_document({ width: 800, height: 600 })
2. affinity_create_layer({ name: "Background", type: "standard" })
3. affinity_select_tool({ tool: "ellipse" })
4. affinity_create_shape({ shape: "ellipse", width: 200, height: 200 })
5. affinity_create_layer({ name: "Text Layer" })
6. affinity_add_text({ text: "TechCorp", fontSize: 36 })
7. affinity_apply_effect({ effect: "drop_shadow", intensity: 0.7 })
8. affinity_export_document({ format: "png", quality: 100 })
