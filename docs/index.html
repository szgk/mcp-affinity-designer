Quick Start

クイックスタート

1

Install the Package

パッケージをインストール

npm install mcp-affinity-designer
2

Configure MCP Client

MCPクライアントを設定

{
  "mcpServers": {
    "affinity-designer": {
      "command": "node",
      "args": ["./node_modules/mcp-affinity-designer/dist/index.js"]
    }
  }
}
3

Start Using

Available Tools

利用可能なツール

Application Management

  • affinity_get_status - Check application status
  • affinity_start_app - Launch Affinity Designer
  • affinity_focus_window - Focus application window

Document Operations

  • affinity_new_document - Create new documents
  • affinity_save_document - Save documents
  • affinity_export_document - Export to various formats
  • affinity_close_document - Close documents

Layer Management

  • affinity_list_layers - View layers panel
  • affinity_create_layer - Create new layers
  • affinity_delete_layer - Delete selected layers
  • affinity_rename_layer - Rename layers

Object Operations

  • affinity_select_all - Select all objects
  • affinity_move_selection - Move selected objects
  • affinity_resize_selection - Resize selected objects
  • affinity_duplicate_selection - Duplicate objects
