🚀 Getting Started

Learn how to install and configure the MCP server for your AI assistant.
AIアシスタント用のMCPサーバーのインストールと設定方法を学習。

⚠️ Prerequisites: Install Affinity Designer using MSI installer from official website to ensure proper installation in C:\Program Files\.
⚠️ 必要条件: 公式サイトからMSIインストーラーを使用してAffinity Designerをインストールし、C:\Program Files\への適切なインストールを確保してください。

Installation Commands / インストールコマンド:

# Clone the repository / リポジトリをクローン
git clone https://github.com/szgk/mcp-affinity-designer.git
cd mcp-affinity-designer

# Install dependencies / 依存関係をインストール
npm install

# Build the project / プロジェクトをビルド
npm run build
  1. Clone the GitHub repository using the command above
  2. Install dependencies with npm install
  3. Build the project with npm run build
  4. Configure your MCP client
  5. Verify Affinity Designer is installed
  6. Configure custom paths in affinity-install-paths.json if needed
  7. Test basic functionality

⚙️ Custom Installation Paths

Configure custom Affinity Designer installation paths.
カスタムAffinity Designerインストールパスを設定。

📝 Configuration File: Edit affinity-install-paths.json in the project root to add your custom installation paths.
📝 設定ファイル: プロジェクトルートのaffinity-install-paths.jsonを編集してカスタムインストールパスを追加。

Example Configuration / 設定例:

{
  "windows": {
    "affinityInstallPaths": [
      "C:\\Program Files\\Affinity\\Designer 2\\Designer.exe",
      "C:\\Program Files\\Affinity\\Affinity Designer 2\\Affinity Designer 2.exe",
      "D:\\Custom\\Affinity Designer\\Designer.exe"
    ]
  },
  "macos": {
    "affinityInstallPaths": [
      "/Applications/Affinity Designer 2.app/Contents/MacOS/Affinity Designer 2",
      "/Applications/Affinity Designer.app/Contents/MacOS/Affinity Designer",
      "/Custom/Path/Affinity Designer.app/Contents/MacOS/Affinity Designer"
    ]
  }
}
  1. Locate the JSON file in your project root directory
  2. Add your custom path to the appropriate platform section
  3. Use full executable paths (including .exe on Windows)
  4. Use double backslashes for Windows paths
  5. Save and restart your MCP server
